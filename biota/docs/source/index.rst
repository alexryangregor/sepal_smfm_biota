Documentation for SMFM biota
============================

.. figure:: images/alos_examples.png
   :scale: 50 %
   :align: center

The BIOmass Tool for Alos (``biota``) is a Python library that uses data from JAXA's ALOS mosaic product to generate maps of:

* Aboveground biomass
* Forest cover
* Forest cover change
